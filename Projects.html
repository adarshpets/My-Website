Machine Learning AI

In this project I built a machine learning ai in C++ that takes in all the paizza posts from last semesters and learns from it, and uses that knowledge to predict with topics future piazza posts are about. By implementing the Map and BST Libabry I was able to create a classifier that can classify Piazza posts based on there topics. The classifier built was a simplified version of a “Multi-Variate Bernoulli Naive Bayes Classifier” and used probolity to predict the topics of future Piazza posts. Through the use of this method I was able to make the classifer about 83-87% accurate.

Computer Vision

In this project I built a image resizing program in C++ that makes any image smaller using the seam-curving algorithm. The algroithm works by finidng and removing "seams" in image that pass through the lease important pixels. By making a Matrix and Image ATD that keeps track of the image pixels I was able to compute the energy and the value of the matrix of each pixel in matrix and find the minimal cost seam and remove it.
